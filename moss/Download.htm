Download (Moss interpreter)

Microsoft Windows

  1. Download »moss-win.zip«. SHA-256: »9e80 548f 757e 69e1 3291 10d5 7a9b 3c42 7fb5 4996 f839 d36f 4a4b 8510 8778 93e8«.
  2. Verify the SHA-256-hash by an external trusted source.
  3. Open »moss.exe« in a Windows command-line ([SHIFT]+[right mouse click]: »open command window here«).
  4. Type in »moss« or »moss.exe«. Type in »[CTRL]+[c]« to exit.
  5. Type in »moss m.txt« or »moss m.moss« to execute a module.
