Download (Moss interpreter)

Microsoft Windows

  1. Download "moss-win.zip". SHA-256: 9e80 548f 757e 69e1 3291 10d5 7a9b 3c42 7fb5 4996 f839 d36f 4a4b 8510 8778 93e8.
  2. Verify the SHA-256-hash by an external trusted source.
  3. Open a Windows command-line inside of the unpacked directory ([SHIFT]+[right mouse click]: "open command window here").
  4. Type in "moss" or "moss.exe". Type in "[CTRL]+[c]" to exit.
  5. Type in "moss m.txt" or "moss m.moss" to execute a module.
  6. Move moss.exe to a directory that is found in environment variable PATH.

Linux

A portable application (x86), but with reduced functionality:

  1. Download "moss-linux-portable.tar.bz2". SHA-256: 4763 ccfa 92b5 056a f1d1 46fe a0d2 1443 f5db 6532 2a3a 029b e58e 3272 8c6e 8fb0.
  2. Verify the SHA-256-hash by an external trusted source.
  3. Open a terminal window inside of the unpacked directory.
  4. Type in "./moss". Type in "[CTRL]+[c]" to exit.
  5. Type in "./moss m.moss" to execute a module.
  6. Move the binary file "moss" to /usr/local/bin/ or any other directory that is found in environment variable PATH ("echo $PATH").
