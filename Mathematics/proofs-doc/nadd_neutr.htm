Theorem. nadd_neutr
a ∈ ℕ → a + 0 = a
Proof
01. 1 ⊢ a ∈ ℕ, hypo.
02. 1 ⊢ a ∈ ℝ, incl_elim nat_incl_in_real 1.
03. 1 ⊢ a + 0 = a, radd_neutr 2.
nadd_neutr. ⊢ a ∈ ℕ → a + 0 = a, subj_intro 3.

Dependencies
The given proof depends on five axioms:
comp, eq_subst, radd_closed, radd_neutr, real_is_set.
