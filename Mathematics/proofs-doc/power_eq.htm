Definition. power_eq
power M = {A | A ⊆ M}

The power set of a set is the set of all its subclasses. According to the axiom of power sets it can never be a proper class. The power of the universal class is the universal class itself, as shown in power_UnivCl. This strange behavior can be explained in terms of universes. The true power, that is power M, is replaced by a cut-off version U ∩ power M where U is a universe, a set with M ⊆ U that resembles the universal class.
