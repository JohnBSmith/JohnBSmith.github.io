Theorem. onat_induction_sets
A ⊆ onat → (∅ ∈ A ∧ ∀n. n ∈ A → succ n ∈ A) → A = onat
Proof
01. 1 ⊢ A ⊆ onat, hypo.
02. 2 ⊢ ∅ ∈ A ∧ ∀n. n ∈ A → succ n ∈ A, hypo.
03. 2 ⊢ ∅ ∈ A, conj_eliml 2.
04. 2 ⊢ ∀n. n ∈ A → succ n ∈ A, conj_elimr 2.
05. 5 ⊢ n ∈ A, hypo.
06. 2, 5 ⊢ succ n ∈ A, uq_bounded_elim 4 5.
07. 2, 5 ⊢ n ∪ {n} ∈ A, eq_subst succ_eq 6, P x ↔ x ∈ A.
08. 2 ⊢ n ∈ A → n ∪ {n} ∈ A, subj_intro 7.
09. 2 ⊢ ∀n. n ∈ A → n ∪ {n} ∈ A, uq_intro 8.
10. 2 ⊢ ∅ ∈ A ∧ ∀n. n ∈ A → n ∪ {n} ∈ A, conj_intro 3 9.
11. 1 ⊢ set A, subset 1 onat_is_set.
12. 1, 2 ⊢ A ∈ {A | ∅ ∈ A ∧ ∀n. n ∈ A → n ∪ {n} ∈ A}, comp_intro 11 10.
13. 1, 2 ⊢ A ∈ onat_inductive_sets,
  eq_subst_rev onat_inductive_sets_eq 12, P x ↔ A ∈ x.
14. 1, 2 ⊢ onat ⊆ A, Intersection_is_lower_bound onat_eq 13.
15. 1, 2 ⊢ A = onat, incl_antisym 1 14.
onat_induction_sets. ⊢ A ⊆ onat →
  (∅ ∈ A ∧ ∀n. n ∈ A → succ n ∈ A) → A = onat, subj_intro_ii 15.

Dependencies
The given proof depends on six axioms:
comp, eq_refl, eq_subst, ext, infinity, subset.
