Theorem. univ_closed_pair_set
univ U → x ∈ U → y ∈ U → {x, y} ∈ U
Proof
01. 1 ⊢ univ U, hypo.
02. 1 ⊢ (∀x. ∀y. x ∈ U → y ∈ U → {x, y} ∈ U), lsubj_conj_elimllr univ_equi 1.
03. 1 ⊢ ∀y. x ∈ U → y ∈ U → {x, y} ∈ U, uq_elim 2.
04. 1 ⊢ x ∈ U → y ∈ U → {x, y} ∈ U, uq_elim 3.
univ_closed_pair_set. ⊢ univ U → x ∈ U → y ∈ U → {x, y} ∈ U, subj_intro 4.
