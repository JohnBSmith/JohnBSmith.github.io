Theorem. nadd_neutl
a ∈ ℕ → 0 + a = a
Proof
01. 1 ⊢ a ∈ ℕ, hypo.
02. 1 ⊢ a ∈ ℝ, incl_elim nat_incl_in_real 1.
03. 1 ⊢ 0 + a = a, radd_neutl 2.
nadd_neutl. ⊢ a ∈ ℕ → 0 + a = a, subj_intro 3.

Dependencies
The given proof depends on seven axioms:
comp, eq_refl, eq_subst, radd_closed, radd_comm, radd_neutr, real_is_set.
