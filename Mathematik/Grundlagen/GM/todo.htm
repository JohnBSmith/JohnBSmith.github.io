Manche Leute neigen dazu, Aussagen, von deren Wahrheitsgehalt sie nicht überzeugt sind, aus Überlegungen herauszuhalten. Ob die Prämissen von Argumenten tatsächlich wahr sind, darf aber beim logischen Schließen keine Rolle spielen. Zum Beispiel möchten wir eine Konklusion in ein oder mehreren Argumenten aus Annahmen, deren Wahrheitsgehalt noch unklar bleibt, ableiten dürfen. Konnte hierbei ein Widerspruch abgeleitet werden, so ist im Umkehrschluss klar, dass mindestens eine der Annahmen falsch sein muss. Hätten wir falsche Aussagen als Prämissen disqualifiziert, so hätten wir diese Überlegung nicht durchführen können, uns bliebe eine Erkenntnis verwehrt.
